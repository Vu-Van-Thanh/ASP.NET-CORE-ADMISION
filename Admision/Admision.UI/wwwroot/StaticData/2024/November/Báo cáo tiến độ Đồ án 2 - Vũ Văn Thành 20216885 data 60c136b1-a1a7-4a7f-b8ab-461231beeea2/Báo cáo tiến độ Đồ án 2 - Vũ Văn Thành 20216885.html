-          Chủ để :Xây dựng ứng dụng tuyển sinh bằng ASP.NET Core

-          Đã hoàn thành :

  • Khảo sát hệ thống cũ ,(ưu, nhược điểm, yêu cầu, cải tiến hệ thống mới)
  • Mô hình hóa nghiệp vụ bằng sơ đồ PBMN và diễn giải bằng lời
  • Sơ đồ usecase cùng phân rã sơ đồ usecase
  • Đặc tả , xây dựng sơ đồ hoạt động, sơ đồ tuần tự một số usecase
  • Thiết kế cơ sở dữ liệu : xác định lớp khái niệm,lược đồ ER , biểu đồ lớp(đang hoàn thiện qua từng hiện thực hóa usecase) , domain
  • Cài đặt :

      Front-end : trang chủ , đăng nhập/đăng kí /đổi mật khẩu/ đăng kí thông tin tuyển sinh/blog chat/ tin tức tuyển sinh/Đăng bài tuyển sinh (admin)/Đăng bài viết hỏi đáp(user)

      Back-end : trang chủ/ đăng nhập/đăng kí/đổi mật khẩu/Đăng thông tin tuyển sinh (admin) bằng aspose.NET/ Triển khai database với SQL Server/Crawl Data bằng Python Scrapy

-          Mục tiêu tiếp theo:

  • Xây dựng được trang xem tin tức việc làm thông qua dữ liệu đã được crawl
  • Xây dựng power BI
  • Hoàn thành chức năng BlogChat cho mỗi khoa/trường
  • Tích hợp hệ thống thanh toán vào thi tuyển sinh
  • Tinh chỉnh lại giao diện
